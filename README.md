# Business Operations Backend

A backend API built with Python, FastAPI, and PostgreSQL to manage customers, products, invoices, and core business operations.

## Overview

This project is designed as a practical backend system for small business operations. It focuses on core modules such as authentication, product management, customer management, and invoice handling.

The goal of this project is to reflect a real-world backend architecture with clean organization, scalable structure, and business-oriented logic.

## Features

- User authentication
- Product management
- Customer management
- Invoice creation
- PostgreSQL integration
- REST API structure
- Modular backend organization

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Pydantic
- Uvicorn
